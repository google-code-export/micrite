-------------------------------------------------------------------------------
概述
-------------------------------------------------------------------------------

Gaixie Micrite是一个基于Struts2,Spring2的完整项目实现。


-------------------------------------------------------------------------------
编译
-------------------------------------------------------------------------------

1. 编译Micrite需要的环境:

 * JDK 5.0+ (J2SE 1.5.0+) (http://java.sun.com/j2se/1.5.0)
 * Maven 2.0.5+ (http://maven.apache.org)

2. 编译Micrite:

    mvn install
   
    *从头进行一次干净的编译，运行下面命令
    mvn clean install
    
详细编译指南见：http://code.google.com/p/micrite/wiki/Building
-------------------------------------------------------------------------------
运行
-------------------------------------------------------------------------------

1. 运行Micrite业务模块:

 * 进入Micrite/module/业务模块,然后运行下面命令
   mvn jetty:run

2. 运行整个Micrite项目:

 * 进入Micrite/assemblies，运行下面命令
    mvn jetty:run

-------------------------------------------------------------------------------
文档
-------------------------------------------------------------------------------
1.生成项目文档

  *进入项目根目录，运行下面命令
  mvn site package
  
  会在assemblies/taget下产生micrite-site目录，此目录存放所有项目相关文档（包括apidocs）

2.官方文档
详细Micrite相关文档请访问 http://code.google.com/p/micrite/w/list
