-------------------------------------------------------------------------------
概述
-------------------------------------------------------------------------------

Gaixie Micrite是一个基于Struts2,Spring2的完整项目实现。


-------------------------------------------------------------------------------
编译
-------------------------------------------------------------------------------

1. 编译Micrite需要的环境:

 * JDK 5.0+ (J2SE 1.5.0+) (http://java.sun.com/j2se/1.5.0)
 * Maven 2.0.5+ (http://maven.apache.org)

2. 编译Micrite:

    mvn install
   
从头进行一次干净的编译.

    mvn clean install
